fastcomposer-layouts-fastcampus
===============================

fastcomposer layouts for fastcampus.
### install dependencies

```console
$ npm install
```

### watch & build (scss files only)

```console
$ npm run watch
```

will generates:
* `src/**/style.css` from `src/**/style.scss`
